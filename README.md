# DEVELOPMENT

## Install dep

```
git clone https://github.com/nhancv/stakingpooljs
cd stakingpooljs
yarn install
```

## Local dev

- Normal mode - without monitor

```
yarn start
```

- Monitor mode

```
yarn start:dev
```

## Build production

```
yarn build
```

## Test

- Run test

```
yarn test
```

- Run test coverage

```
yarn test:cov
```
